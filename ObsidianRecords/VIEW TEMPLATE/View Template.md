# View Engine
- 서버에서 뷰를 구현해주는 엔진

# View Template
- 뷰를 만드는데 사용되는 템플릿 (html, xml이나 다른 마크업 언어)
- 개별 문법을 사용할 수 있다

# View Template Engine
- 템플릿([[JSP]],[[Thymeleaf]]) 내의 코드를 해석해서 뷰에 최종적용하는 역할

### JSP와 Thymeleaf의 차이점
- **jsp**: 자바 기반 템플릿 엔진. 자바코드와 html을 혼합하여 조건문,반복문 등 데이터처리
- **Thymeleaf** : html의 구조를 해치지 않고 타임리프만의 문법을 사용하여 조건 처리가능
